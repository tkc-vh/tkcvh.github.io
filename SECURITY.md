# Security Policy

## Overview

This repository hosts a **static GitHub Pages website** for academic, literary, and archival purposes.

- No server-side code
- No databases
- No user authentication
- No data collection or tracking
- No executable uploads

All content is delivered as static HTML, CSS, and related assets via GitHub Pages.

---

## Supported Versions

As this is a static website, there are no versioned releases requiring security patching.

The site content is continuously maintained through GitHub commits.

## Scope of Security Reports

Valid security reports include:
- Unexpected redirects
- Content integrity issues
- Unauthorized repository modifications
- Phishing or impersonation attempts involving this domain

The following are **out of scope**:
- False positives from antivirus software without technical evidence
- Issues related to third-party platforms linked from the site
- Browser-specific rendering issues

---

## Responsible Disclosure

We appreciate responsible disclosure and request that reporters:
- Do not exploit vulnerabilities beyond proof of concept
- Do not publicly disclose issues before resolution
- Allow reasonable time for review and response

## Transparency Statement

This website is:
- **Static**
- **Openly hosted on GitHub Pages**
- **Free of malware, trackers, and hidden scripts**

Security reports are welcomed and taken seriously.
